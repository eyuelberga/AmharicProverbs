Amharic Proverbs, a Vue3 web application to access Amharic proverbs from the Amharic Proverbs API.

## Features

- Random proverbs
- Share proverbs on Twitter and Facebook
- Search

### Installation

```bash
$ yarn
```

### Running the app

```bash

$ yarn dev
```
